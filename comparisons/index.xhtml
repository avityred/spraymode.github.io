Comparisons

Web browsers

tl;dr: Your best bet is GNU Icecat but I also recommend Abrowser or Palemoon with a few tweaks.


VPN providers

tl;dr: Don't use one, just use TOR or I2P. If you still need a VPN then go with IVPN or Mullvad.


Search engines

tl;dr: Avoid proprietary search engines, instead use SearXNG or Whoogle.


Instant messaging services

tl;dr: Just use Matrix or IRC.


Git services

tl;dr: For self hosting I recommend cgit, else use sourcehut or Codeberg.
